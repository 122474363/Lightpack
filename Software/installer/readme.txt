To build installer:
- Install InnoSetup
- Copy libgcc_s_dw2-1.dll, Prismatik.exe, mingwm10.dll, QtCore4.dll, QtGui4.dll, QtNetwork4.dll and other libs to content directory don't forget to build Prismatik without D3D10_GRAB_SUPPORT and place it to content/below_win7 
- Copy plugins to content/plugins
- Open InnoSetup Compiler
- Press Ctrl+F9
- See Output dir.
